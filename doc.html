About

  • This app displays the current weather of any common places all over the world. It displays the humidity, temperature, cloud precipitation, the sunrise and sunset timing and so on.Moreover it also shows weather information of some common cities all across the globe in the below section of the app. The app is designed using HTML, CSS and javascript using the RapidAPI and Live Server VS code extension to fetch live information from the web.
  • The API used for this app is "weather-by-api-ninjas". The link for the API is weatherAPI
